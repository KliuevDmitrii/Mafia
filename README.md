# Mafia

### Шаги
1. Склонировать проект 'git clone https://github.com/KliuevDmitrii/Mafia.git'
2. Установить зависимости
3. Запустить тесты 'pytest'

### Стек:
- pytest
- selenium
- allure
- config

### Струткура:
- ./test - тесты
- ./pages - описание страниц
